
# Problem
Given...
    - Int, numBottles
    - Int, numExchanges

Return...
    - Int

Where...
numBottles is the initial number of full water bottles, numExchanges is the 
number of empty water bottles you can turn in for 1 full water bottles. But,
every time you perform an exchange, numExchanges will increment by 1. Return
the max number you water bottles you can drink.

[\#3100 Water Bottles II](https://leetcode.com/problems/water-bottles-ii/description/?envType=daily-question&envId=2025-10-02)

## Constraint
- 1 <= numBottles <= 100
- 1 <= numExchanges <= 100
